# Sobek
Simple 40% ortolinear keyboard (inspired by [Plank Keyboard](https://olkb.com/collections/planck)).
Uses Pro Micro or [nrfMicro](https://github.com/joric/nrfmicro/wiki) controller (for wireless bluetooth connection).
